# Playlist Creator v1.0

Creates a m3u file from a folder with a mp3s files. Uses mp3 ID tags to set the song name and artist.

## Usage

java -jar playlistcreator-1.0-jar-with-dependencies.jar mp3-files-directory 

Example:
java -jar playlistcreator-1.0-jar-with-dependencies.jar /Users/alvaro/msk/Rock
